About me


Hi everyone, I'm Filip, a software developer from Belgrade, the capital of Serbia. I was born on March 6, 1986, in Belgrade, making me 38 years old. Growing up in the analog age, before the rise of commercial internet, I was always fascinated by video games and the digital world. I dreamed of understanding how it all worked, but the limited information and development at that time made it difficult.

I watched older kids with their 286, 386, and 486 computers and longed for one of my own. In 1998, when I was in fifth grade of Elementary School, I got my first computer, a Pentium. I was overjoyed and immediately dove into video games and writing word documents. I remember the first ones was "Civilization" and "Championship Manager". My parents regularly gave me tasks to type and format word documents for them. In the same year I started playing basketball and competing in younger categories (I earned my first medal in 2001). That how I found another passion that continues to this day. I experienced the internet for the first time a year later. It was Dial-up internet, using a telephone line and audio signals for data transmission. This system was quite inefficient, taking a long time to connect, data packet loss, frequently dropping connections due to phone calls...

I went to the 4th Belgrade Gymnasium School. My direction was natural sciences and the most successful subjects were mathematics, logic and sports. This is where I encountered programming for the first time. The first programming languages ​​I met were Java and Pascal. Here I met a lot of good people and many of them are now my friends. Events and anecdotes continue to be told. At that time, there were popular PC gaming rooms where we often went and played in LAN. We even created our own virtual servers via dial up and played from home. I mostly played Warcraft, Counter Strike, Dota and Pro Evolution Soccer.

After finishing high school, I entered the Faculty of Transport and Traffic Engineering in Belgrade. I passed the entrance exam in Mathematics without any mistakes with the maximum number of points. I would throw the next two years into oblivion. Puberty and dating left me without passing the exam and I decided to leave college. But there is something good in every experience. I decided to enroll in a new college and choose a new profession.

In 2007 I entered Singidunum University, Faculty of Bussines and Informatics (later renamed in Faculty of Informatics and Computing). I have chosen the programming and design direction. I studied various subjects, but the most relevant are Basics of Programming, Java, Web programming, Databases, Economics, Bussines, Management, Math, Statistic... At the same time, I joined the basketball team and represented my university at various sports tournaments in the largest European cities. I brought a lot of medals and made new friends from these competitions. I graduated in 2012 and earned the "Bachelor's Degree of Business Informatics".

JavaScript:

                                
function calculateAge() {
    const dob = document.getElementById('dob').value;
    const birthHour = parseInt(document.getElementById('birth-hour').value);
    const birthMinute = parseInt(document.getElementById('birth-minute').value);
    
    if (dob && !isNaN(birthHour) && !isNaN(birthMinute) && !(birthHour>24) && !(birthMinute>60)) {
        const dobDate = new Date(dob);
        dobDate.setHours(birthHour, birthMinute, 0, 0);  // Set hour and minute of birth

    const today = new Date();
    
    // Check if the date of birth is in the future
    if (dobDate > today) {
        document.getElementById('result').innerText = 'The date of birth cannot be in the future.';
        return;
    }

    // Calculate years
    let ageYears = today.getFullYear() - dobDate.getFullYear();
    let ageMonths = today.getMonth() - dobDate.getMonth();
    let ageDays = today.getDate() - dobDate.getDate();
    let ageHours = today.getHours() - dobDate.getHours();
    let ageMinutes = today.getMinutes() - dobDate.getMinutes();
    let ageSeconds = today.getSeconds() - dobDate.getSeconds();

    // Adjust months and years
    if (ageMonths < 0 || (ageMonths === 0 && ageDays < 0)) {
        ageYears--;
        ageMonths += 12;
    }

    // Adjust days and months
    if (ageDays < 0) {
        ageMonths--;
        let lastMonth = (today.getMonth() - 1 + 12) % 12;
        let daysInLastMonth = new Date(today.getFullYear(), lastMonth + 1, 0).getDate();
        ageDays += daysInLastMonth;
    }

    // Adjust hours and days
    if (ageHours < 0) {
        ageDays--;
        ageHours += 24;
    }

    // Adjust minutes and hours
    if (ageMinutes < 0) {
        ageHours--;
        ageMinutes += 60;
    }

    // Adjust seconds and minutes
    if (ageSeconds < 0) {
        ageMinutes--;
        ageSeconds += 60;
    }

    document.getElementById('result').innerText = `Your age is: 
        ${ageYears} years, 
        ${ageMonths} months, 
        ${ageDays} days, 
        ${ageHours} hours, 
        ${ageMinutes} minutes, 
        and ${ageSeconds} seconds old.`;
    } else {
        document.getElementById('result').innerText = 'Please enter a valid date of birth, hour, and minute.';
    }
}
